# tinkle — Landing Page Design Direction

> North star: cream + sky + blue ocean, slightly painterly. Think a Monet painting. Skeuomorphic but not too much.

## References
- **Chapters grid** — cream base, subtle grid, soft layered cards, monospace labels, pill CTA.
- **Tomo** — playful skeuomorphic object-collage, sky / clouds / grass, bold rounded display, tactile bubble icons.
- **About / orb** — grainy blurred gradient (sky → ocean → sunset), serif display, atmospheric and quiet.

## Texture & finish
- Film grain / noise over everything, subtle.
- Soft blurred radial gradients that bleed sky → ocean → sunset (the Monet move).
- Skeuomorphism = soft depth, glassy bubbles, gentle shadows. Tactile, not glossy.

## Type
- **Serif display** — emotional / editorial headlines ("Find the stories that connect us.").
- **Clean sans (grotesque)** — UI + bold statements ("Live your life to the fullest").
- **Monospace** — labels, metadata, microcopy ("Chapter I", "2026", "by Cofounder").

## Components
- Soft rounded cards, generous radius, subtle layered shadow; stacked / peeking card effect.
- Pill buttons with arrow or icon ("Build my world →").
- Glassy bubble icons for nav.
- Optional signature: tactile object-collage as a hero device (Tomo's phone number).
- Background: subtle grid OR grainy gradient orb — not both at once.

## "Not too much" — guardrails
| Move | Yes | No |
|---|---|---|
| Skeuomorphic | soft depth, glassy bubbles, gentle shadows | heavy bevels, glossy Web 2.0, shadow overload |
| Painterly | grain + soft gradient bleeds | literal brushstrokes, busy painted scenes |
| Nature / sky | atmospheric color & light | photo landscapes wall-to-wall |
| Color | cream base, blue-led, accents sparingly | saturated everywhere |
